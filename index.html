Have you ever thought to yourself: "Self, I really need to make a change, do better for myself"? You have this thought and that is all it ends up being, just a thought. What if you could put that thought into some kind of tracker, and calendar? Here we give you Habits 66! A full MERN web application.

Our Main Goal

To develop an application to help you keep track of new habits you want to form. You want to wake up early and set a new schedule for yourself? Then go on the app and add it to your new list of habits. Set a time period of how long you feel it will take you before you pick up the habit.

Authentication API

"The Okta Authentication API provides operations to authenticate users, perform multi-factor enrollment and verification, recover forgotten passwords, and unlock accounts. It can be used as a standalone API to provide the identity layer on top of your existing application, or it can be integrated with the Okta Sessions API to obtain an Okta session cookie and access apps within Okta."

In order to use the application, you must first register and create an account.

Once you create an account, you can then use your credentials to sign in and start creating and tracking your habits.

react-calendar

Ultimate Calendar for your react app: You can pick days, months, years and even decades; Supports range selection; Supports virtually any language; and No moment.js needed. We integrated this into the application as the calendar to help keep track of how long you've been maintaining your new found habit.

Inspirational Quotes

"A simple NPM Package which returns random Inspirational Quotes. It provides awesome quotes to display in your application. Get your daily quote and stay motivated!" This is integrated into the application to keep you motivated everytime you log on and visit the habits page.
